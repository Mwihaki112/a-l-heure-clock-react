## Project Description
a l’heure` is a dynamic digital clock built with React.
It shows the current time and date in a stylish, French-inspired interface.
The app updates every second and is designed with pastel colors and elegant fonts.

The name “a` l’heure” means “the hour” or “the time” in French, and in context can also imply being on time, reflecting the app’s purpose.

## Features
- Displays current time (HH:mm:ss) updating every second.
- Shows current date (Day, Month Date Year).
- Elegant, centered layout suitable for desktop and mobile.
- French-inspired pastel gradient background.
- Styled boxes with rounded corners and shadows.
- Subtle pulse animation on the time display.

## Technologies Used
- React
- JavaScript
- date-fns (for formatting date and time)
- CSS (for styling)

## Setup Instructions
1. Clone the repository:
git clone
2. Navigate to the project folder:
cd a-lheure
3. Install dependencies:
npm install
4. Start the development server:
npm run dev
5. Open the URL shown in the terminal (usually http://localhost:5173) in your browser.

## How to Use
1. Open the app in a browser.
2. The current time and date are displayed automatically.
3. The time updates every second.
4. No additional input is required — it’s fully automatic.

## Color Palette
1. Background gradient: #f6e8e0 → #e0f6f1 
2. Title: #34495e (slate blue/gray)
3. Time box: #2c3e50 (deep contrast color)
4. Date box: #7f8c8d (light accent gray)
5. Fonts: Monospace for time, sans-serif for title/date
